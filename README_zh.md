EZ Translate Lite 是一款轻量级浏览器翻译插件，利用大型语言模型（LLM）提供准确且贴合上下文的翻译体验。此版本是原始 EZ Translate 项目的分支，经过简化和优化，提供更好的可用性。

**注意：这是一个简化版本** - 我们移除了多个复杂功能，创建了一个更专注、更轻量级的翻译工具，同时保留了核心功能。

## 关于此分支

此项目派生自 [licon/ez-translate](https://github.com/licon/ez-translate)，主要变更如下：

### 🗑️ **移除的功能（简化）**
- **固定的供应商限制**：移除了对特定供应商（Gemini、SiliconFlow、OpenRouter）的硬编码支持
- **语音/朗读功能**：移除了文字转语音功能
- **右键菜单翻译**：简化了翻译流程
- **截图/图片翻译**：移除了基于图片的翻译功能
- **复杂的供应商特定配置**：简化了设置

### ✅ **增强的功能**
- **灵活的供应商支持**：现在支持任何 LLM 供应商，可自定义 API 端点
- **可拖动的翻译弹窗**：翻译窗口可以在屏幕上任意移动
- **直接文本复制**：翻译结果中的文本可以直接选中和复制
- **简化的用户界面**：更简洁、更专注的用户界面
- **轻量级设计**：减少了扩展大小和复杂度

### 🎯 **核心理念**
此分支优先考虑简单性和可用性，而不是功能完整性。目标是提供一个快速、可靠的翻译工具，可以与任何 LLM 供应商配合使用，无需不必要的复杂性。

## 安装

### 手动安装（推荐）
如果您无法访问浏览器扩展商店，推荐使用此方法。

1. 从 GitHub Releases 下载最新压缩包：[:link: 最新版本](https://github.com/Abelliuxl/ez-translate-lite/releases/latest)
2. 解压下载的 `.zip` 文件
3. **Chrome/Edge/Brave（Chromium 内核）**：
   - 打开 `chrome://extensions`（Edge: `edge://extensions`，Brave: `brave://extensions`）
   - 打开「开发者模式」
   - 点击「加载已解压的扩展程序」，选择解压后包含 `manifest.json` 的文件夹
4. **Firefox**：
   - 打开 `about:debugging#/runtime/this-firefox`
   - 点击「临时载入附加组件…」，选择解压目录中的 `manifest.json`
5. **更新方法**：重新下载最新 `.zip`，解压后在扩展页点击「重新加载」

## 核心功能

*   **灵活的供应商支持**：配置任何 LLM 供应商，支持自定义 API 端点。不再局限于特定供应商。
*   **划词翻译**：在任何网页上选择文本，翻译图标会出现。点击即可在原位查看翻译结果。
*   **弹窗翻译**：点击浏览器工具栏图标打开翻译弹窗。输入或粘贴文本进行即时翻译。
*   **可拖动的弹窗**：翻译弹窗可以在屏幕上任意拖动和重新定位。
*   **直接文本选择**：翻译结果可以直接选中和复制，无需额外按钮。
*   **智能自动填充**：打开弹窗时，选中的文本会自动填充到输入框。
*   **一键复制**：内置复制按钮，可快速复制翻译结果。
*   **安全本地存储**：API 密钥存储在本地浏览器中，绝不上传。
*   **智能语言检测**：自动检测源语言，并在需要时切换目标语言。
*   **多语言支持**：支持 100+ 种语言的翻译。

## 翻译方式

### 📍 **划词翻译**
1. **选中文字** - 在任意网页上选择文本
2. **翻译图标出现** - 在选中区域附近显示
3. **点击图标** - 查看翻译结果
4. **复制或交互** - 根据需要进行操作

### 🔲 **弹窗翻译**
1. **点击扩展图标** - 在浏览器工具栏中
2. **输入或粘贴文本** - 在输入框中
3. **点击翻译** - 获取翻译结果
4. **拖动弹窗** - 将其重新定位到屏幕上的任何位置
5. **选中并复制** - 直接从结果中选中和复制文本

### ⚡ **智能特性**
- **自动填充**：选中的文本自动填充到弹窗输入框
- **语言检测**：自动在目标语言间切换
- **可拖动界面**：将翻译弹窗移动到屏幕上的任何位置
- **直接文本访问**：直接选中和复制翻译结果

## 配置

### 供应商设置
此分支支持灵活的供应商配置：

1. 打开扩展设置
2. 选择您偏好的 LLM 供应商类型
3. 输入您的 API 端点和密钥
4. 根据需要配置模型参数

### 支持的供应商类型
- **自定义 API 端点**：使用任何具有兼容 API 的 LLM 供应商
- **OpenAI 兼容的 API**：支持提供 OpenAI 兼容接口的服务
- **本地模型**：配置本地推理服务器

## 技术栈

*   **前端**：`HTML`、`CSS`、`JavaScript`
*   **浏览器接口**：`WebExtensions API`（兼容 Chrome、Firefox、Edge 等）
*   **存储**：`chrome.storage.local` 用于安全的本地数据存储

## 与原始版本对比

| 功能 | 原始 EZ Translate | 此分支 |
|------|------------------|--------|
| 供应商支持 | 固定供应商（Gemini、SiliconFlow 等） | **灵活，支持任何供应商** |
| 翻译方式 | 划词、弹窗、右键、截图 | **仅划词、弹窗**（简化版） |
| 界面复杂度 | 多个功能和选项 | **简化、专注的界面** |
| 弹窗行为 | 固定位置 | **可拖动、可重新定位** |
| 文本复制 | 仅复制按钮 | **直接选择 + 复制按钮** |
| 语音功能 | 文字转语音支持 | **为简化而移除** |
| 图片翻译 | 截图翻译 | **为简化而移除** |

## 为什么创建此分支？

原始的 EZ Translate 是一个功能丰富的扩展，但一些用户更喜欢更简单、更专注的工具。此分支：

1. **降低复杂度**：移除不常用的功能
2. **增加灵活性**：支持任何 LLM 供应商
3. **提高可用性**：可拖动的弹窗和直接文本复制
4. **保留核心功能**：满足日常翻译需求
5. **轻量级设计**：性能更优
